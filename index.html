Cybor's Virtual Mouse Documentation

Cybor's Virtual Mouse is an advanced gesture-controlled virtual mouse system that revolutionizes human-computer interaction using cutting-edge Computer Vision and Machine Learning technologies.

System Requirements

Hardware

  • Webcam (720p minimum, 1080p recommended)
  • Microphone for voice commands
  • 4GB RAM minimum (8GB recommended)
  • Intel i5 or AMD Ryzen 5 equivalent

Software

  • Python 3.8.5+
  • MediaPipe Framework
  • OpenCV
  • PyAudio for voice processing

Installation Guide

1

Clone Repository

git clone https://github.com/abhishek-tiwary/cybor-virtual-mouse.git
2

Create Environment

conda create --name cybor-mouse python=3.8.5
3

Activate Environment

conda activate cybor-mouse
4

Install Dependencies

pip install -r requirements.txt
5

Run Cybor System

python Cybor.py

Technical Architecture

Computer Vision Pipeline

Utilizes MediaPipe's hand detection and landmark estimation for precise finger tracking and gesture recognition with sub-pixel accuracy.

Machine Learning Models

Custom CNN models trained on diverse hand gesture datasets, optimized for real-time inference with TensorFlow Lite.

Voice Processing

Advanced speech recognition using PyAudio and custom wake word detection for the "Cybor" activation command.

System Integration

Native OS integration for mouse control, system commands, and application interaction across Windows, macOS, and Linux.
